# MiniLearn Bloom 🌸

Application web front-end de suivi d'habitudes réalisée dans le cadre du TP DWWM.

## Description

MiniLearn Bloom permet de suivre ses habitudes quotidiennes et de visualiser sa progression sous forme d'un jardin qui pousse 🌱.

### Fonctionnalités

- ✅ **Gestion d'objectifs** : Ajoutez, cochez et supprimez vos objectifs quotidiens
- 🔥 **Suivi de séries** : Visualisez vos séries de jours consécutifs
- 🏆 **Système de badges** : Déverrouillez des badges de constance (3, 7, 14, 30 jours)
- 🌱 **Jardin virtuel** : Votre jardin pousse au fur et à mesure de vos accomplissements
- 🌙 **Mode clair/sombre** : Basculez entre les thèmes selon vos préférences
- 💾 **Sauvegarde locale** : Toutes vos données sont sauvegardées dans le localStorage

## Technologies utilisées

- **HTML5** : Structure sémantique de l'application
- **CSS3** : Design responsive avec variables CSS et animations
- **JavaScript (Vanilla)** : Logique dynamique et gestion du localStorage
- **localStorage** : Persistence des données sans base de données

## Installation et utilisation

1. Clonez ce repository
2. Ouvrez le fichier `index.html` dans votre navigateur
3. Commencez à ajouter vos objectifs et à cultiver votre jardin!

## Compétences DWWM couvertes

- ✅ Environnement de développement front-end
- ✅ Création de maquettes/interfaces statiques
- ✅ Développement d'interface statique (HTML/CSS)
- ✅ Développement de logique dynamique (JavaScript)
- ✅ Gestion de données côté client (localStorage)

## Aperçu des fonctionnalités

### Badges de constance
- 🌱 **Débutant** : 3 jours consécutifs
- 🌿 **Constant** : 7 jours consécutifs
- 🌳 **Dévoué** : 14 jours consécutifs
- 🌸 **Maître** : 30 jours consécutifs

### Jardin virtuel
Chaque objectif complété fait pousser une nouvelle plante dans votre jardin. Plus vous êtes constant, plus votre jardin devient luxuriant!
